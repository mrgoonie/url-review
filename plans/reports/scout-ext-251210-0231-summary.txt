API ENDPOINTS (16 MODULES)
================================================================================

CORE FEATURES:
  POST   /api/v1/review              - Create website review
  GET    /api/v1/review              - List reviews (paginated)
  GET    /api/v1/review/{reviewId}   - Get single review
  PATCH  /api/v1/review/{reviewId}   - Update review
  DELETE /api/v1/review/{reviewId}   - Delete review

  POST   /api/v1/scrape              - Scrape single URL
  POST   /api/v1/scrape/urls         - Batch scrape (max 10 URLs)
  POST   /api/v1/scrape/links-map    - Extract & analyze links

  POST   /api/v1/extract             - AI-powered data extraction
  GET    /api/v1/ai/models           - Get available AI models

  POST   /api/v1/screenshot          - Capture screenshots
  GET    /api/v1/seo-insights        - SEO analysis
  POST   /api/v1/summarize           - Content summarization
  POST   /api/v1/convert             - URL conversion

USER & ACCOUNT:
  GET    /api/v1/profile             - Get user profile
  GET    /api/v1/api_key             - List API keys
  POST   /api/v1/api_key             - Create API key
  DELETE /api/v1/api_key/{id}        - Delete API key

BILLING:
  POST   /api/v1/payments            - Process payment
  POST   /api/v1/order               - Create order

SYSTEM:
  GET    /api/v1/healthz             - Health check
  POST   /api/v1/upload              - File upload

WEBHOOKS:
  /webhooks/polar                    - Polar payment webhooks
  /webhooks/standard                 - Generic webhooks
